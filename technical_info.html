Specifications

Secure storage and secure transmission of data integrate many ideas which often need to be composed separately in different applications. Data provenance capabilities add the need for further composable components. We make an effort to breakout ideas into independent specifications so that they can be integrated as appropriate for different projects.

BluMamba Patent
BluMamba is the codename assigned to the concept of cryptographically dividing data into shares and managing the state transitions necessary for data disassembly and data reassembly. This aims to distribute these shares across different geographical locations.
PinkScorpion Patent
PinkScorpion is the codename for the data disassembly and reassembly protocol, supported by a state machine operated by a blockchain. The blockchain serves as evidence of the integrity of the data disassembly process and any subsequent reassembly process.
A vision for integrated Data Authentication and Data Integrity
Data Authentication is the widely accepted concept for confirming, without a doubt, the origin of data. It entails validating the source of data to guarantee that it remains unaltered and originates from a trustworthy entity.
